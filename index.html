Merhaba, ben Cem. patika.dev ve kodluyoruz.org eğitimlerini takip ederek front-end öğreniyorum.

Patika.dev HTML Ödev 1

Git ile sayfamda yaptığım değişiklikleri tüm detayları ile takip edebiliyorum. Değişiklik yaptığımda güncel kodlarımı uzak sunucuya gönderip projemi güncelleyebilirken aynı zamanda istediğimde projemin eski haline de ulaşabiliyorum. Ekip arkadaşlarımla farklı noktalardan aynı proje üzerinde çalışabiliyorum. GitHub ve muadili sitelerle başkalarının paylaşıma açık olarak yüklediği projeleri inceleyebiliyorum, kendi bilgisayarıma indirebiliyorum veya ben başkaları ile projemi paylaşabiliyorum.

HTML ile basit web sayfaları tasarlanılabiyorken daha ileri seviye sayfaların omurgaları yine HTML ile oluşturuluyor, CSS ve Java Script ile de göze hitap eden işlevli sayfalar tasarlanılabiliyor. Başlık ve paragraf kodlarını öğrendim. Açıklama kodunu öğrendim, projenin anlaşılır olabilmesi açısından açıklama yazmak önemli. Özellikle uzun kod dizileri bulunan projelerde daha önce çalıştığım kısımları bulabilmek için, takım arkadaşlarımın benim projemi rahat inceleyebilmesi veya onların yaptıklarını benim kolay takip edebilmem için açıklama satırı kullanmalıyım.
